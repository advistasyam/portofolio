Penggunaan teknologi informasi sudah menjadi hal yang lumrah pada masyarakat dengan menggunakan banyak media, contohnya aplikasi web. Aplikasi web digunakan oleh banyak orang dengan sektor yang berbeda. Sektor tersebut adalah lain perdagangan / e-commerce seperti Tokopedia dan e-learning seperti contoh aplikasi web Scele yang digunakan mahasiswa Fasilkom UI. Dalam penggunaannya aplikasi web harus memiliki performa yang baik ketika digunakan banyak orang sehingga tidak mengganggu proses kerja yang terjadi pada aplikasi. Selain performa dibutuhkan aplikasi web teroptimisasi dalam mesin pencarian / SEO (Search Engine Optimisation). Terdapat kerangka kerja web yang mendukung performa dan SEO yang baik, contohnya adalah NextJs.

# Apa Itu NextJS ?

NextJs adalah kerangka kerja yang digunakan oleh pengembang untuk membuat aplikasi web. Next Js berbasiskan library React Js sehingga pengguna yang telah terbiasa membuat aplikasi menggunakan React Js tidak akan kesulitan menggunakan NextJs. Selain itu juga NextJs menggunakan bahasa pemrograman javascript agar dapat beroperasi. NextJs diciptakan oleh tim pengembang bernama Vercel dan dapat digunakan oleh pengembang secara gratis. Pengembang yang baru memulai dapat menggunakan dokumentasi yang disediakan oleh Vercel untuk belajar menggunakan kerangka kerjanya dan dapat mengunggah aplikasi yang dibuat di server milik Vercel.

## Keuntungan

![img](/static/images/blog/next_img2.webp)

Kerangka kerja NextJs memiliki banyak keuntungan yang ditawarkan untuk pengguna aplikasi web. Dalam hal performa, NextJs menawarkan opsi yang meningkatkan kecepatan memuat aplikasi, contohnya :

- Next Image, membuat ukuran bit gambar menjadi lebih kecil
- SSG (Static Site Generator) yang memungkinkan browser pengguna untuk memuat file tanpa harus menunggu server merender file tersebut.
- Hanya ada 1 HTML file untuk setiap request pada web
  NextJs juga menawarkan pengembang untuk mengatur kata kunci pencarian sehingga meningkatkan nilai SEO aplikasi menggunakan fitur Next SEO.

## Kekurangan

Walaupun memiliki banyak keuntungan, terdapat beberapa kontra yang dimiliki oleh NextJs. Komunitas yang dimiliki NextJs masih lebih sedikit dibanding kerangka kerja lain seperti Laravel, hal ini disebabkan NextJs termasuk kerangka kerja yang baru muncul pada tahun 2016. NextJs belum menawarkan State Management data yang terkait sehingga pengembang perlu menggunakan library lain seperti redux. NextJs adalah kerangka kerja aplikasi web yang kaya fitur sehingga meningkatkan performa aplikasi dan nilai SEO. Penggunaan NextJs membuat aplikasi dapat digunakan oleh banyak orang dengan lebih baik dan tidak mengganggu kelancaran proses bisnis yang terjadi. Dengan fitur yang ditawarkan Pengembang juga harus memperhatikan kekurangan yang dimiliki dan mencari solusi penyelesaiannya.

# Memulai menggunakan NextJS

![img](/static/images/blog/next_img3.webp)

Vercel sebagai pencipta framework tersebut membuat module yang dapat digunakan oleh developer untuk membuat suatu boilerplate sebelum memulai menulis kode. Selain itu juga terdapat beberapa tools yang diharus terinstall pada komputer dimiliki yaitu

- NodeJS, dicek melalui node -version pada command line
- npm, dicek melalui npm -version pada command line
  Apabila kedua hal tersebut, pengguna dapat membuka folder kosong tujuan dan menjalankan

```bash
npm install -g create-next-app
```

Tunggu hingga proses selesai dan jalankan

```bash
npx create-next-app
```

Setelah setelah NextJS dapat dijalankan pada server developement menggunakan command

```bash
npm run dev
```

Dan aplikasi dapat dilihat pada http://localhost:3000
